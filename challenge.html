GIT CHECKOUT

What does Git checkout do?

Git checkout is a git command used to move to another branch that has been formerly created

Using git checkout NAME_0F_BRANCH, we can migrate to the other branch created.

This command can also be used to remove files from the working directory before they are added to the staging area

Using git checkout NAME_OF_FILE Note that this command can not be undone so use carefully.

Flag that can be used with Git checkout:

The -b flag is used with git checkout to create a new branch using this command:git checkout -b NAME_OF_BRANCH

Hence,Git checkout works hand in hand with Git branch.
